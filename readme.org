Collection of custom commands for Revit. Solves common challenges encountered at technical delivery of buildings.

** supported versions

| Revit Version | Status        | Notes                                   |
|---------------+---------------+-----------------------------------------|
| 2022-2026     | Full support  | All commands available                  |
| 2021          | Full support  | All commands available*                 |
| 2020          | Mostly works  | 29 commands excluded                    |
| 2019          | Mostly works  | 29 commands excluded                    |
| 2017-2018     | Mostly works  | 31 commands excluded                    |

*Revit 2017-2020: Remote scripting server requires manual SSL certificate generation.
Run in PowerShell: ~New-SelfSignedCertificate -DnsName 'localhost' -CertStoreLocation 'Cert:\CurrentUser\My'~

Excluded commands in older versions primarily use newer APIs for:
- PDF export (2022+)
- Enhanced tag management (2022+)
- Unit system APIs (2021+)
