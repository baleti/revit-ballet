#+TITLE: Command Standardization Plan
#+DATE: 2026-04-10
#+STATUS: DRAFT — awaiting review before execution

* Goal

Align all commands with the canonical conventions described in
[[file:architecture-vision.org]], specifically:

1. *No "Selected" in the class name* — selection is implied behaviour
2. *Singular form* — commands operate on one or many inputs the same way
3. *Standard input resolution* — use active view / filter selection / DataGrid
   picker depending on what's in the selection and what the command expects

The changes below are *renames only* (no logic changes) unless noted otherwise.
Every renamed class must also update:
- The source ~.cs~ file name (keep 1:1 with class name)
- Any cross-file references (currently ~ViewDuplicator~, ~SheetDuplicator~, etc.)
- The ribbon button entry in ~CommandRegistry.cs~ / ~*.addin~ manifest / ribbon XML
  (verify before executing)

* Input Resolution Standard

Implement a shared helper before or alongside these renames:

#+BEGIN_SRC csharp
// Proposed: commands/InputResolver.cs
public static class InputResolver
{
    public static List<T> ResolveViews<T>(UIDocument uiDoc)
        where T : View { ... }

    public static List<ViewSheet> ResolveSheets(UIDocument uiDoc) { ... }

    public static List<Element> ResolveElements(UIDocument uiDoc,
        params BuiltInCategory[] categories) { ... }
}
#+END_SRC

Rules (from [[file:architecture-vision.org]]):
- View/Viewport input + nothing selected → active view
- View/Viewport input + selection → filter to matching type
- Sheet + nothing selected → active view if it's a sheet
- Other element type → DataGrid picker if nothing relevant selected
- Always silently filter mixed selections to the relevant type

* Naming Convention

| Old pattern                  | New pattern         | Example                              |
|------------------------------+---------------------+--------------------------------------|
| ~FooSelectedBars~            | ~FooBars~           | ~DuplicateSelectedViews~ → ~DuplicateView~ |
| ~FooSelectedBar~             | ~FooBar~            | ~CopyCropRegionOfSelectedView~ → ~CopyCropRegion~ |
| ~FooBars~ (plural)           | ~FooBar~            | ~DuplicateViewsAsDraftingViews~ → ~DuplicateViewAsDraftingView~ |
| ~FooSelected~ at end         | ~Foo~               | ~ZoomSelected~ → ~ZoomToSelection~ (or ~ZoomView~) |
